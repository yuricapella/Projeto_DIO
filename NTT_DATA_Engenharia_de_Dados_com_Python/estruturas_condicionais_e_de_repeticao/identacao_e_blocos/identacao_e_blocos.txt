# A estética

Identar código é uma forma de manter o código fonte mais
legível e facil de fazer manutenções depois.

Mas em Python ela exerce um segundo papel, através da indentação o interpretador consegue
determinar onde um bloco de comando inicia e onde ele termina.

# Utilizando espaços

Existe uma convenção em Python, que define as boas práticas
para escrita de código na linguagem. 
Nesse documento é indicado utilizar 4 espaços em branco por nível de indentação,
ou seja, a cada novo bloco adicionamos 4 novos espaços em branco.
4 espaços ou apertar tab no VSCode

# Como deve escrever:
def sacar(self, valor: float)->None: # inicio do bloco do método

  if self.saldo valor: # inicio do bloco do if

    self.saldo -= valor

  # fim do bloco do if

# fim do bloco do método



# Assim o código não vai funcionar

if self.saldo valor: # inicio do bloco do if
self.saldo -= valor

#fim do bloco do método
